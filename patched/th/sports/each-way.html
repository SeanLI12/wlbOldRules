กฎการเดิมพัน “ทายอันดับ”

อัพเดทล่าสุด: 15/12/2017

การเดิมพันแบบ “ทายติดอันดับ”  ประกอบไปด้วย 2 ตัวเลือกในหนึ่งบิล  (ด้วยจำนวนเงินที่เท่ากันในแต่ละบิล) ; คือ ทาย “ชนะ” และ “ติดอันดับ”

  1. หากตัวเลือกของท่าน “ชนะ” จะได้รับเงินชนะเดิมพันจากทั้ง 2 ตัวเลือก
  2. หากตัวเลือกของท่าน “แพ้” ท่านยังมีโอกาสได้รับเงินชนะ หากอีกตัวเลือกหนึ่งติดอันดับ

เมื่อระบบยืนยันบิลเดิมพันของท่าน จะแสดงผลเป็น 2 ส่วน คือ “ชนะ” และ “อันดับ”

แต่ละตลาดใช้จำนวนอันดับที่ไม่เหมือนกัน  (อันดับในตารางและราคาคูณ)  และจำนวนอันดับจะเป็นตัวบ่งชี้ราคาเงินที่ชนะเดิมพัน

ราคาทายเดิมพัน “ติดอันดับ” ที่จะเห็นโดยทั่วไปอย่างเช่น  ¼ - ติดหนึ่งใน 3  หมายความว่า  ทีมที่เลือกจะติดหนึ่งใน 3 อันดับเมื่อจบการแข่งขัน  และหากเหตุการณ์นี้เป็นจริง ท่านจะได้เงินชนะเดิมพัน (ครึ่งหนึ่งของเงินเดิมพันทั้งหมด) เป็น  ¼ ของราคาคูณตัวเลือกที่ชนะ จำนวน”อันดับ” ที่ใช้ในแต่ละเดิมพันสามารถเป็นตัวเลขอื่นได้ เช่น 1/5 – ติดหนึ่งใน 5  หรือ 1/5 – ติดหนึ่งใน 7 เป็นต้น

ผลในบิลเดิมพันแบบ “ทายอันดับ” สามารถเป็นได้ดังนี้

  1. “ชนะ” โดย ตัวเลือกที่ทายนั้นเข้าอันดับหนึ่ง
  2. “ชนะบางส่วน” โดย ตัวเลือกที่ทายติดในอันดับที่กำหนดแต่ไม่ใช่ที่หนึ่ง
  3. “แพ้”  ตัวเลือกที่ทายไม่เข้าและไม่ติดอันดับที่กำหนดเลย

ตัวอย่างทั้งหมดของแต่ละข้อจะอยู่ใต้กฎทั่วไป

กฎทั่วไป:
  • เดิมพัน “ทายติดอันดับ” เปิดให้วางเดิมพันคู่กับตลาด “ทายผลทีมชนะ”  เท่านั้น โดยใช้กฎของกีฬาแต่ละประเภทในการตัดสิน
  • ข้อกำหนดเดิมพันแบบ “ทายติดอันดับ”  (อันดับ และ ราคาคูณ) อาจมีการเปลี่ยนแปลงเมื่อการแข่งขันเริ่มดำเนิน เช่น การแข่งขันกอล์ฟ ให้ยึด อันดับ และราคาคูณ  (จากบิลเดิมพันของท่าน)
  • การเดิมพันแบบ “ทายติดอันดับ” อาจเป็นโมฆะ ถ้าจำนวนผู้เข้าร่วมตั้งแต่เริ่มต้นการแข่งขัน มีการเปลี่ยนแปลง ตัวอย่างเช่น  มีผู้เข้าร่วมที่กำหนดไว้ 5 คน แต่เกิดเหตุการณ์ ทำให้เหลือเพียง 3 คน ก่อนที่การการแข่งขันจะเริ่มต้น
  • การเดิมพันสเต็ป/มัลติเพิ้ล แบบ “ทายอันดับ (เช่น 2 ทีม, 3 ทีม และ 4 ทีม) สามารถวางเดิมพันได้หากทั้งหมดที่เลือกในสลิปการวางเดิมพันเปิดให้วางเดิมพันแบบ “ทายติดอันดับ”
  • การตัดสินการวางเดิมพัน “ทายติดอันดับ”  แบบสเต็ป/มัลติเพิ้ล จะใช้ผลเพียงสองอย่างคือ  ผล  “ชนะ กับ ชนะ” และ “ติดอันดับ กับ ติดอันดับ”  (ไม่ใช่การวมของทั้งสอง)  ตัวอย่างเช่น หากเลือกสเต็ป 2 ( ติดอันดับ  2 ทีม) ผลคือ  ชนะ 1 และติดอันดับ 1 หากเลือกวางเดิมพัน “ชนะ กับ ชนะ” จะถือว่าแพ้  แต่ถ้าวางเดิมพัน “ติดอันดับ กับ ติดอันดับ” จะถือว่าชนะ
  • การตัดสินการวางเดิมพัน “ทายติดอันดับ”  และ  “ทายชนะ”  (หรือ ทายชนะ เท่านั้น) การวางเดิมพันจะขึ้นอยู้กับกฎเดทฮีท

    ตัวอย่าง A : เลือกเดิมพัน “ชนะ” (จบในอันดับที่ 1)

    วางเดิมพันทายทีมชนะ  –  อังกฤษพรีเมียร์ลีก  (ทายทีมชนะ)

    ทีมที่เลือก : อาเซน่อล @ 13.00  I   วางเดิมพัน : 100.00   I   เงื่อนไขเดิมพัน ทายอันดับ – ราคาคูณ ¼ , ติดหนึ่งใน 3

    เดิมพันทั้งหมด = 200.00 (รวมยอดวางเดิมพัน “ทายติดอันดับ”)

    ผล : อาเซน่อล เป็นทีมที่ชนะอันดับ 1 ของลีก

    ชนะ →  13.0 (ราคาคูณ) X 100 (เงินเดิมพัน) = 1,200.00

    ติดอันดับ →  4.0 (ราคาคูณทายติดอันดับ) X 100 (เงินเดิมพัน) = 300.00

    ชนะ / แพ้  คือ  1,200 + 300 = + 1,500.00


    ตัวอย่าง B : เลือกเดิมพัน “อันดับ” (จบติดหนึ่งใน 3 )

    เดิมพันทายทีมชนะ – อังกฤษพรีเมียร์ลีก  (ทายทีมชนะ)

    ทีมที่เลือก : อาเซน่อล @ 13.00  I   วางเดิมพัน : 100.00   I   เงื่อนไขเดิมพัน ทายอันดับ – ราคาต่อรอง ¼ , ติดหนึ่งใน 3

    เดิมพันทั้งหมด = 200.00 (รวมยอดวางเดิมพัน “ทายติดอันดับ”)

    ผล : อาเซน่อล  ได้ตำแหน่งอันดับ 3 ของลีก

    ชนะ →  อาเซน่อล ไม่ได้ชนะอันดับที่ 1 ของลีก;  ดังนั้นการวางเดิมพันจะเสีย  = - 100.00

    ติดอันดับ →  4.0 (ราคาคูณทายติดอันดับ) X 100 (เงินเดิมพัน) = 300.00

    ชนะ / แพ้  คือ   -100 + 300 = + 200.00


    ตัวอย่าง C : การเดิมพันแพ้ ( ไม่ติดหนึ่งใน 3)

    เดิมพันทายทีมชนะ – อังกฤษพรีเมียร์ลีก  (ทายทีมชนะ)

    ทีมที่เลือก : อาเซน่อล @ 13.00  I   วางเดิมพัน : 100.00   I   เงื่อนไขเดิมพัน ทายอันดับ – ราคาคูณ ¼ , ติดหนึ่งใน 3

    เดิมพันทั้งหมด = 200.00 (รวมยอดวางเดิมพัน “ทายติดอันดับ”)

    ผล : อาเซน่อล  ได้อันดับที่ 4 ของลีก

    ชนะ →  อาเซน่อล ไม่ได้ชนะอันดับที่ 1 ของลีก;  ดังนั้นการวางเดิมพันจะเสีย  = - 100.00

    ติดอันดับ →  อาเซน่อล ไม่ติดหนึ่งใน 3 ของลีก ; ดังนั้นการวางเดิมพันเสีย  =  - 100.00

    ชนะ / แพ้  คือ   -100 + -100 =  -200.00
