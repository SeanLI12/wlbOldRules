កីឡាវាយប៉េងប៉ុង

កាលបរិច្ឆេទចុងក្រោយ: 29/03/2022

ក្បួនទូទៅ
  • រាល់ការភ្នាល់ទាំងអស់ នឹងត្រូវចាត់ទុកថាមានសុពលភាព នៅពេលដែលការប្រកួត ឬអំឡុងពេលជាក់លាក់ ត្រូវបានលេងរហូតដល់ចប់ លើកលែងតែមានការបញ្ជាក់នៅក្នុងច្បាប់ប្រភេទភ្នាល់ឯកត្តជន
  • ប្រសិនបើទម្រង់នៃការដាក់ពិន្ទុនៃការប្រកួតមួយ មានភាពផ្សេងគ្នាពីទម្រង់ដើមដែលត្រូវបានបញ្ជាក់ រាល់ការភ្នាល់ទាំងអស់ នឹងត្រូវចាត់ទុកជាមោឃៈ។
  • ប្រសិនបើការប្រកួតមួយ ចាប់ផ្តើមលេង នៅមុនពេលកម្មវិធីដែលគ្រោងទុក មានតែអន្តរការណ៍នានា នៅមុនពេលការចាប់ផ្តើមរាប់ពិន្ទុទីមួយប៉ុណ្ណោះ ត្រូវបានចាត់ទុកថាមានសុពលភាព។ អន្តរការណ៍នានា បន្ទាប់ពីពិន្ទុទីមួយនៃការប្រកួត នឹងត្រូវចាត់ទុកថាគ្មានសុពលភាព។ (លើកលែងតែប្រភេទភ្នាល់ពេលកំពុងលេង)។
ប្រភេទនៃការភ្នាល់

ជ័យលាភី

  • ទាយថាអ្នកណានឹងឈ្នះការប្រកួត។

ហ្គេម គ្រាប់ត

  • ទាយថាអ្នកណានឹងឈ្នះ ការប្រកួតដោយមានការកំណត់គ្រាប់ត/ការលែងដោយ គ្រាប់។

ហ្គេមសរុប: លើស / ក្រោម

  • ទស្សន៍ទាយថាតើចំនួននៃសេត/ហ្គេម នឹងលើស រឺក្រោមចំនួនដែលបានបង្ហាញ។

ពិន្ទុគ្រាប់ត

  • ទាយថាអ្នកណានឹងឈ្នះពិន្ទុច្រើនជាងនៅក្នុងអំឡុងពេលជាក់លាក់មួយ ក្រោយពេលដាក់ការគ្រាប់តដែលបាន ណត់ហើយ។

ពិន្ទុសរុប - លើស / ក្រោម

  • ទស្សស៍ទាយថាតើចំនួនសរុបនៃពិន្ទុឈ្នះដោយកីឡាករតែងតាំងមួយនៅក្នុងកំឡុងពេលជាក់លាក់នឹងត្រូវ លើស ឬក្រោម ចំនួនសរុបបានបញ្ជាក់។

ពិន្ទុក្រុម / កីឡាករ - លើស / ក្រោម

  • ទស្សន៍ទាយថាតើចំនួនសរុបនៃពិន្ទុដែលកត់ត្រាដោយ ក្រុមម្ចាស់ផ្ទះ ឬ ក្រុមភ្ញៀវ នឹងត្រូវបានលើស ឬ ក្រោម ពិន្ទុដែលបានកំណត់។

សិត/ការភ្នាល់ហ្គេម (Set / Game Betting)

  • ព្យាករអ្នកឈ្នះសិត/ហ្គេម ជាក់លាក់។

ការប្រកួតទាយគ្រាប់ត្រឹមត្រូវ

  • ទស្សន៍ទាយពិន្ទុត្រឹមត្រូវ ពេញតង់នៃការប្រកួតជាក់លាក់មួយ

សេស/គូ

  • ទស្សន៍ទាយថាតើចំនួនសរុបនៃហ្គេម សេត រឺ ពិន្ទុត្រូវបានលេងក្នុងកំឡុងពេលជាក់លាក់ជាចំនួន គូ រឺ សេស។
