ច្បាប់​នៃ​ការ​ភ្នាល់

កែប្រែចុងក្រោយ: 09/09/2017

 

Each Way មាន​​​ប្រភេទ​នៃ​ការ​ភ្នាល់​2ដាច់​ដោយ​ឡែក​ពី​គ្នា​(ប្រាក់​នៃ​ការ​ភ្នាល់​ស្មើ​គ្នា) ទៅ​លើជំរើសតែ​មួយ​​ ដែល​មាន​ការ​ភ្នាល់​លើ​ក្រុម​ឈ្នះ (Win Bet) និង​ការ​ភ្នាល់​លើ​ចំណាត់​ថ្នាក់ (Place Bet)​។

  1. ប្រសិន​បើ​ជម្រើស​​នេះ​ឈ្នះ ទឹក​ប្រាក់​ភ្នាល់​ទាំង​ពីរ នឹង​ត្រូវ​​សង​ជូន​វិញ បូក​ទាំង​ប្រាក់​ដែល​ត្រូវឈ្នះ​។
  2. ប្រសិន​បើ​ជម្រើស​នេះ​ចាញ់ ទឹកប្រាក់មានកំណត់មួយអាចនឹងត្រូវសង ប្រសិន​បើ​លក្ខ​ខណ្ឌ​នៃ​ការ​ភ្នាល់​លើ​ចំណាត់​ថ្នាក់​ត្រូវបានបំពេញ។

នៅ​ពេល​ការ​ភ្នាល់​ត្រូវ​បាន​បញ្ជាក់ ប្រភេទ​ភ្នាល់​ Each Way នឹង​បង្ហាញ​ចេញ​ជា​ពី​រ​ផ្នែកគឺ ការ​ភ្នាល់​លើ​ក្រុម​ឈ្នះ និង​ការ​ភ្នាល់​​លើ​ចំណាត់​ថ្នាក់​។

​ផ្សារ​ភ្នាល់​នីមួយ​ៗ​មាន​លក្ខខ​ណ្ឌផ្សេងគ្នានៃការភ្នាល់​លើ​ចំណាត់​ថ្នាក់ (លទ្ធផលចំណាត់​ថ្នាក់ និង​តម្លៃ​ហាងឆេង) ហើយ​ការ​ភ្នាល់​នេះ នឹង​កំណត់​ចំនួន​ដែល​ត្រូវ​ឈ្នះ​។

លក្ខខ​ណ្ឌនៃ​ការ​ភ្នាល់​លើចំណាត់​ថ្នាក់ (Place​ terms) ជា​ធម្ម​តាបង្ហាញដូចជា ឧទាហរណ៍ 1/4 -​ ​​ចំណាត់​ថ្នាក់​កំពូល​ទាំង (Top 3)​។ នេះ​មាន​ន័យ​ថា មួយ​ភាគ​បួន​នៃ​តម្លៃ​ហាងឆេង​ដើម នឹង​ត្រូវសង​ជូន​វិញ (​ពាក់​​កណ្តាលនៃប្រាក់​ភ្នាល់​សរុប​) ប្រសិន​បើ​ក្រុម​ដែល​ជ្រើស​រើស​នោះ ប​ញ្ចប់​ក្នុង​ចំណាត់​ថ្នាក់​កំពូល​ទាំង​3នៃ​ព្រឹត្តិ​ការណ៍​មួយ​។ ​លក្ខ​ខណ្ឌ​នៃ​ការ​ភ្នាល់​លើ​ចំណាត់​ថ្នាក់ (Place terms) មាន​ច្រើន​ប្រភេទ​ផ្សេង​គ្នា និង​អាច​​ជា 1/5 - Top 5 ឬ 1/5 - Top 7 ជា​ដើម​។

លទ្ធផល​ដែល​អាច​កើត​ឡើង​​នៃ​ការ​ភ្នាល់ Each Way រួម​មាន៖

  1. ការ​ភ្នាល់​លើ​ក្រុម​ឈ្នះ​ និងការភ្នាល់​លើ​ចំណាត់​ថ្នាក់ លទ្ធផលជម្រើសទាំង2គឺ “ឈ្នះ​” (Both wins)
  2. ជម្រើសការ​ភ្នាល់​លើ​ក្រុម​ឈ្នះ ​លទ្ធផលគឺ ”​ចាញ់” ហើយ​ជម្រើសការ​ភ្នាល់​លើ​ចំណាត់​ថ្នាក់ លទ្ធផលគឺ ”​ឈ្នះ”
  3. ការ​​ភ្នាល់​លើ​ក្រុម​​ឈ្នះ​ និង​ការភ្នាល់​លើ​ចំណាត់​ថ្នាក់ លទ្ធផលជម្រើសទាំង2គឺ “ចាញ់” (Both lose)

ខាង​ក្រោម​នេះ គឺ​ជា​ឧទាហរណ៍​ពេញ​លេញ​​នៃ​ច្បាប់​ភ្នាល់​​ទូទៅ

ច្បាប់ភ្នាល់ទូទៅ:
  • ប្រភេទភ្នាល់ Each Way អាច​ត្រូវ​​ធ្វើ​ឡើង​តែ​​​លើ​តម្លៃ​ហាងឆេង​ទាំងមូល (Outright Markets) ជាក់​លាក់​ប៉ុណ្ណោះ​។ ច្បាប់​ភ្នាល់​​ទូទៅ​នៃ​កីឡា​នីមួយៗ នឹង​ត្រូវ​ប្រើ​ប្រាស់​ដើម្បី​ទូ​ទាត់​លើ​ការ​ភ្នាល់​ទាំង​នេះ​។
  • លក្ខខ​ណ្ឌនៃការភ្នាល់ Each Way (ចំណាត់​ថ្នាក់ និង​តម្លៃ​ហាងឆេង) នឹង​​ប្រែ​ប្រួលទៅតាមដំណើរការនៃ​ព្រឹត្តិ​ការណ៍​មួយ ឧទាហរណ៍​​ដូចជា ព្រឹត្តិការណ៍កីឡា​វាយ​កូន​គោល​។ ប្រភេទភ្នាល់ Each Way បានកំណត់ (ក្នុងរបាយការណ៍របស់អ្នក) នៅពេល​ដែល​​ការ​ភ្នាល់​ត្រូវ​បាន​​ធ្វើ​ឡើង ស្របពេលដែលទី​​ផ្សារភ្នាល់នីមួយ​ៗ បើក​ដំណើរ​ការ​។
  • ប្រភេទភ្នាល់ Each Way អាច​នឹង​ទុក​ជា​មោ​ឃៈ ប្រសិន​បើ​ចំនួន​នៃ​ក្រុម ឬ​អ្នក​ចូល​រួម​នៃ​ព្រឹត្តិ​ការណ៍ មិន​ស្រប​តាម​ទំរង់​នៃ​ទីផ្សារ​របស់ Each Way។ ឧទាហរណ៍ ក្រុម​ចូល​រួម​ដែល​បាន​គ្រោង​ទុក​​ចំនួន​5 ត្រូវ​បាន​កាត់​បន្ថយ​មក​ត្រឹម​3 នៅមុន​ពេល​ដែលព្រឹត្តិ​ការណ៍​ចាប់​ផ្តើម​។
  • ប្រភេទភ្នាល់ Each Way គូវិល ឬ Each Way Multiples (ឧទាហរណ៍ Doubles, Trebles, 4-Folds ជា​ដើម) អាចធ្វើការភ្នាល់បាន លុះត្រាតែ​ជម្រើសភ្នាល់នីមួយៗដែលនៅក្នុងប័ណ្ណភ្នាល់ បានបើកដំណើរការអោយភ្នាល់ Each Way។
  • ការ​ទូ​ទាត់​នៃ​ប្រភេទភ្នាល់ Each Way គូវិល ផ្អែក​ទៅ​លើ​គោល​ការណ៍ Win-to-Win និង Place-to-Place (មិនមែនជាការរួមបញ្ចូលគ្នា​នៃជម្រើស​ទាំង​ពីរ​)​។ ឧទាហរណ៍ ប្រសិន​បើ​ជម្រើស​ 2 របស់​​អ្នក (Each Way Double) មានលទ្ធផល 1 លើ​ការ​ភ្នាល់​ក្រុម​ឈ្នះ​ និង 1 ​លើការ​ភ្នាល់ចំណាត់​ថ្នាក់​​ ការ​​ភ្នាល់ Win-to-Win នឹង​ចាញ់ ខណៈ​ដែល​ការ​ភ្នាល់ Place-to-Place នឹង​ឈ្នះ​។
  • ការ​ទូ​​ទាត់​នៃ​ការ​ភ្នាល់​ក្រុម​ឈ្នះ និង​ចំណាត់​ថ្នាក់​របស់ Each Way (ឬ​ក៏​ការ​ភ្នាល់​តែ​លើ​ចំណាត់​ថ្នាក់) ការភ្នាល់នឹង​ត្រូវ​អនុ​វត្ត​​តាម​ច្បាប់ Dead Heat​។

ឧទាហរណ៍::

  1. ជម្រើសនៃការភ្នាល់ក្រុមឈ្នះ (លទ្ធផលក្នុងចំណាត់ថ្នាក់ទី 1)

    តម្លៃ​ទី​ផ្សារ​ទាំងមូល - ក្រប​ខ័ណ្ឌ​អង់​គ្លេស Premier League (ក្រុម​ឈ្នះ​)

    ជម្រើស ឬ​ក្រុម​ដែល​ជ្រើស​រើស: Arsenal @ 13.0  |   ប្រាក់​ភ្នាល់: 100.00  |  ប្រភេទភ្នាល់ E/W - 1/4 Odds, Top 3

    ទឹក​ប្រាក់​ភ្នាល់​សរុប = 200.00 (inc. Each Way)

    លទ្ធផល: Arsenal ឈ្នះ​ពាន​លីគ

    ក្រុម​​ឈ្នះ​​ (Win) → 13.0 (តម្លៃ​ហាងឆេង​របស់ការភ្នាល់លើក្រុម​ឈ្នះ) X 100 (ទឹក​ប្រាក់​ភ្នាល់​) = 1,200.00

    ចំណាត់​ថ្នាក់ (Place) → 4.0 (តម្លៃ​ហាងឆេង​របស់ការភ្នាល់លើចំណាត់​​ថ្នាក់) X 100 (ទឹក​ប្រាក់​ភ្នាល់​)= 300.00

    ឈ្នះ / ចាញ់ គឺ 1,200 + 300 = + 1,500.00


  2. ជម្រើសនៃការភ្នាល់លើចំណាត់ថ្នាក់ (លទ្ធផលក្នុងចំណាត់ថ្នាក់កំពូលទាំង3 ឬ Top 3)

    តម្លៃ​ទី​ផ្សារ​ទាំងមូល - ក្រប​ខ័ណ្ឌ​អង់គ្លេស Premier League (ក្រុម​ឈ្នះ​)

    ជម្រើស ឬ​ក្រុម​ដែល​ជ្រើស​រើស: Arsenal @ 13.0  |   ប្រាក់​ភ្នាល់: 100.00  |  ប្រភេទភ្នាល់ E/W - 1/4 Odds, Top 3

    ទឹក​ប្រាក់​ភ្នាល់​សរុប = 200.00 (inc. Each Way)

    លទ្ធផល: Arsenal បានជាប់​ក្នុង​ចំណាត់​ថ្នាក់​លេខ​3

    ក្រុម​​ឈ្នះ​ (Win) → Arsenal មិន​បាន​ឈ្នះ​ពាន​លីគ​ឡើយ ហេតុនេះ ​​ការ​ភ្នាល់គឺ​ចាញ់ = -100.00

    ចំណាត់​ថ្នាក់ (Place) → 4.0 (តម្លៃ​ហាងឆេងភ្នាល់​លើ​ចំណាត់​ថ្នាក់) X 100 (ទឹកប្រាក់​ភ្នាល់)​ ​= 300.00

    ឈ្នះ / ចាញ់ គឺ -100 + 300 = + 200.00
