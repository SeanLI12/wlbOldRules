VOLI PANTAI

Terakhir Diperbaharui: 20/09/2016

PERATURAN UMUM
  • Semua taruhan akan dianggap berlaku ketika pertandingan berakhir, kecuali hal lain yang dinyatakan.
  • Jika adanya nama pemain yang diganti pada pertandingan sebelum pertandingan dimulai, perusahaan berhak untuk membatalkan semua taruhan.
  • Hasil dari permainan ini ditentukan berdasarkan pada 2 set pertama dari 3 set bemain akan dianggap sebagai pemenang.
  • Jika durasi jadwal dari pertandingan dikurangi atau adanya penaikann dari angka poin untuk menang, semua taruhan akan dianggap batal.
  • Jika permainan dimulai sebelum jadwal permainan, hanya taruhan yang ditempatkan sebelum permainan dimulai akan dianggap sah. Taruhan yang ditempatkan setelah permainan dimulai akan dianggap batal. Jenis taruhan In-Play tidak termasuk.

JENIS TARUHAN

Pemenang

  • Prediksi siapa yang akan menang dalam taruhan tersebut.

Set Handicap

  • Prediksi siapa yang akan menang dalam permainan / periode tersebut dengan set handicap yang ditunjukkan.

Poin Handicap

  • Prediksi siapa yang akan menang banyak poin pada spesifikasi periode (misalnya Pertandingan, Set Pertama, Set Kedua) setelah indikasi handicap yang ditunjukkan.

Jumlah Poin: Atas / Bawah

  • Prediksikan jumlah angka dari poin pada spesifikasi periode (misalnya Pertandingan, Set Pertama, Set Kedua) akan pada diatas atau dibawah dari jumlah garis yang ditunjukkan.
  • Jika pertandingan ditinggalkan, taruhan Over/Under akan diselesaikan ketika pasar telah ditentukan tanpa syarat dan potensial poin lanjut tidak berpengaruh pada hasil pasar. Jika ada skenario lainnya, taruhan akan dianggap batal.

Odd / Even

  • Jika pertandingan ditinggalkan, taruhan Over/Under akan diselesaikan ketika pasar telah ditentukan tanpa syarat dan potensial poin lanjut tidak berpengaruh pada hasil pasar. Jika ada skenario lainnya, taruhan akan dianggap batal.
