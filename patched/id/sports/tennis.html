Terakhir Diperbaharui: 29/03/2022

PERATURAN UMUM
  • Semua taruhan akan dianggap sah jika pertandingan atau periode tertentu dimainkan sampai selesai kecuali dinyatakan sebaliknya dalam aturan Jenis Taruhan individu
  • Jika format penilaian pertandingan berbeda dari yang ditunjukkan pada awalnya, semua taruhan akan dibatalkan.
  • Jika terjadi perubahan tempat (misalnya, indoor ke outdoor atau sebaliknya) atau permukaan lapangan, sebelum atau selama pertandingan, semua taruhan akan dianggap sah.
  • Jika pertandingan dimulai sebelum jadwal, hanya transaksi yang dipasang sebelum pertandingan dimulai yang dianggap sah. Transaksi yang dipasang setelah pertandingan berlangsung akan dianggap tidak sah (kecuali untuk tipe taruhan in-play)
  • 'Super Tie-Break' dianggap sebagai permainan, untuk tujuan penyelesaian.
TIPE TARUHAN

Pemenang

  • Prediksi siapa yang akan menang dalam taruhan tersebut.

Set Handicap

  • Prediksi siapa yang akan menang dalam permainan tersebut dengan set handicap yang ditunjukkan.

Game Handicap

  • Prediksi siapa yang akan menang dalam permainan dalam spesifikasi periode (misalnya pertandingan Set Pertama, Set Kedua) setelah ditunjukkan indikasi handicap.

Pemenang Game – Set (X) Game (X)

  • Tebak siapa yang akan menang di game tertentu dari set tertentu (e.g. Set 1 Game 1).

Jumlah Games: Atas / Bawah

  • Prediksi siapa yang akan menang dalam permainan dalam spesifikasi periode (misalnya pertandingan Set Pertama, Set Kedua) akan menjadi diatas (over) atau dibawah (under) jumlah garis yang diindikasikan.

Odd / Even

  • Prediksi siapa yang akan menang dalam permainan dalam spesifikasi periode (misalnya pertandingan Set Pertama, Set Kedua) akan menjadi ganjil (odd) atau genap (even).

Set Betting Markets

  • Prediksikan hasil akhir dari spesifikasi set.

Koreksi Skor Pertandingan

  • Prediksikan hasil akhir set dari pertandingan.

Jumlah Poin – Atas /Bawah

  • Memprediksi apakah jumlah poin yang dimenangkan oleh pemain tunggal atau pemain ganda dalam suatu pertandingan hasilnya di atas atau di bawah total line yang ditunjukkan.
