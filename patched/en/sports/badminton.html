General Rules
  • All bets will be considered valid only when the match or specified period is played to completion unless otherwise stated in the individual Bet Type rules.
  • If the scoring format of a match differs from what was originally indicated, all bets will be void.
  • If a match starts before schedule, only the transactions before the first point of the match will be considered valid. Transactions after the first point of the match will be considered invalid. (Except in-play bet types).
Bet Types

Winner

  • Predict who will win the match.

Point Winner – Game X, Point X

  • Predict who will win a specific point of a specific game (e.g. Game 1, Point 1).

Game X- Race to X Points

  • Predict which participant will first reach the stated number of points within a game (e.g. Game 1, Race to 5).

Set / Game - Handicap

  • Predict who will win the match with the indicated set / game handicap applied.

Set / Game - Over / Under

  • Predict whether the number of sets / games will be over or under the indicated number.

Point Handicap

  • Predict who will win more points in a specified period after applying the indicated point handicap.

Total Points - Over / Under

  • Predict whether the total number of points won by a nominated player in a specific period will be over or under the indicated total line.

Set / Game Betting

  • Predict the outcome of the specified Set / Game.

Player / Team Points - Over / Under

  • Predict whether the total number of points recorded by the home or away team will be over or under the indicated line.

Match Correct Score

  • Predict the Correct Score at Full Time of a specified match.

Odd / Even

  • Predict whether the total number of games, sets or points played in a specific period will be odd or even.
