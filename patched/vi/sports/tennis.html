QUẦN VỢT

Cập nhật lần cuối: 29/03/2022

LUẬT CHUNG
  • Cược chỉ hợp lệ khi trận đấu hoặc thời gian xác định hoàn tất trừ khi có quy định khác trong luật chơi của từng Loại Cược cụ thể.
  • Nếu cách tính điểm của trận đấu khác với quy định trước đó, tất cả cược đặt sẽ bị hủy.
  • Nếu loại sân thi đấu thay đổi (ví dụ: từ sân trong nhà sang ngoài trời hoặc ngược lại) hoặc mặt sân, trước hoặc trong trận, cược vẫn hợp lệ.
  • Nếu trận đấu diễn ra sớm hơn lịch đấu chính thức, chỉ những cược đặt trước khi trận đấu diễn ra mới được xem là hợp lệ. Những cược đặt sau khi trận đấu diễn ra sẽ bị hủy (trừ cược Trong Trận).
  • Vì mục đích tính cược, lượt đấu 'Super Tie-Break' sẽ được tính là 1 game.
CÁC LOẠI CƯỢC

Đội Vô Địch

  • Cược đặt cho tay vợt thắng cuộc.

Cược Chấp Set

  • Cược đặt cho tay vợt thắng cuộc, có bao gồm tỉ lệ chấp Set.

Cược Chấp Game

  • Cược đặt cho tay vợt thắng nhiều game hơn trong một khoảng thời gian xác định (ví dụ Cả trận, Set 1, Set 2), có bao gồm tỉ lệ chấp.

Thắng Game – Set (X) Game (X)

  • Dự đoán tay vợt sẽ thắng một game cụ thể trong một set cụ thể (ví dụ: Set 1 Game 1).

Tổng Số Game: Trên / Dưới

  • Cược đặt cho tổng số game trong một thời gian xác định (ví dụ Cả trận, Set 1, Set 2) là trên hoặc dưới một hệ số cho trước.

Tổng Số Game: Lẻ / Chẵn

  • Cược đặt cho tổng số game trong một thời gian xác định (ví dụ Cả trận, Set 1, Set 2) là số chẵn hoặc lẻ.

Cược Set Đấu

  • Cược đặt cho kết quả của một Set đấu cụ thể.

Kết Quả Đúng

  • Cược dự đoán kết quả tỉ số set đấu chính xác của trận đấu.

Tổng Số Điểm – Trên/Dưới

  • Cược đặt cho tổng số điểm một tay vợt hay cặp thi đấu ghi được trong một trận là trên hay dưới một hệ số cho trước.
