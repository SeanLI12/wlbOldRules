배드민턴

최근 업데이트: 29/03/2022

일반 규칙
  • 개별 베팅 유형 규칙에 달리 명시되지 않는 한 모든 베팅은 경기 또는 지정된 기간이 완료될 때만 유효한 것으로 간주됩니다.
  • 경기의 득점 형식이 원래 표시된 것과 다른 경우 모든 베팅은 무효화됩니다.
  • 경기가 일정보다 먼저 시작되면 첫 번째 포인트가 시작되기 전의 베팅만 유효한 것으로 간주됩니다. 경기의 첫 번째 포인트 이후의 베팅은 유효하지 않은 것으로 간주됩니다. (인플레이 베팅 유형 제외).
베팅 종류

승리

  • 해당 경기의 승리팀을 예측하는 베팅입니다.

포인트 승리 – 게임 X, 포인트 X

  • 마켓에 명시된 특정 게임에서 특정 포인트로 우승하는 팀을 예측하는 베팅입니다. (예: 게임 1, 포인트 1)

게임 X – X 포인트 선취

  • 게임 진행 동안 명시된 포인트를 먼저 획득하는 승리 팀을 예측하는 베팅입니다. (예: 게임1, 5포인트 선취)

세트 / 게임 – 핸디캡

  • 마켓에 명시된 세트/게임 핸디캡을 적용한 결과를 바탕으로 승리팀을 예측하는 베팅입니다.

세트 / 게임 - 오버 / 언더

  • 진행된 세트 / 게임의 총 수가 주어진 베팅 라인에서 오버 또는 언더일 지를 예측하는 베팅입니다.

포인트 핸디캡

  • 마켓에 명시된 특정 피리어드에서, 주어진 포인트 핸디캡을 적용한 결과를 바탕으로 승리팀을 예측하는 베팅입니다.

총 포인트 – 오버 / 언더

  • 마켓에 명시된 특정 피리어드에서, 지정된 선수로 부터 기록된 총 포인트가 주어진 베팅 라인에서 오버 또는 언더일 지를 예측하는 베팅입니다.

세트 / 게임 베팅

  • 지정된 세트/게임의 승자를 예측합니다.

선수 / 팀 포인트 - 오버 / 언더

  • 홈 또는 원정팀이 기록한 총 포인트가 주어진 베팅 라인에서 오버 또는 언더일 지를 예측하는 베팅입니다.

정확한 스코어

  • 마켓에 명시된 경기의 풀타임 스코어를 예측하는 베팅입니다.

홀 / 짝

  • 마켓에 명시된 특정 피리어드 동안 기록된 게임, 세트 또는 포인트의 총 합이 홀수 또는 짝수일 지를 예측하는 베팅입니다.
