バトミントン

更新日: 29/03/2022

一般規則
  • 個々のベットで特に明記されていない限り、全てのベットは指定された期間または試合が最後までプレーされた場合にのみ有効と見なされます。
  • 試合の得点形式が最初に示されたものと異なる場合、全てのベットは無効になります。
  • 試合が予定時間より前に開始された場合、その最初の得点前のベットのみが有効と見なされます。 試合の最初の得点以降のベットは無効と見なされます。（インプレーでのベットは除く。）
ベットタイプ

勝者

  • 試合で誰が勝つかを予想します。

ポイント勝者 – ゲーム X, ポイント X

  • 特定ゲームにて特定ポイントを誰が勝利するかを予想します（例：ゲーム1、ポイント1）。

ゲーム X - X ポイントまでのレース

  • どの参加者がゲーム内で規定のポイント数に最初に到達するかを予想します（例:ゲーム1、5ポイントまでのレース)。

セット / ゲーム － ハンディキャップ

  • ハンディキャップが適応された特定のセット / ゲームで誰が勝つかを予想します。

セット / ゲーム － オーバー / アンダー

  • セット数 / ゲーム数が指定された数字を超えるか、超えないかを予想します。

ポイント・ハンディキャップ

  • ハンディキャップのポイントが適応された後、特定期間に誰がもっとも得点するかを予想します。

トータル・ポイント － オーバー / アンダー

  • 特定の期間で指定されたプレイヤーの合計得点数が指定された数字を超えるか、超えないかを予想します。

セット / ゲーム・ベッティング

  • 指定されたセット/ゲームの勝者を予測します。

プレイヤー / チーム・ポイント － オーバー / アンダー

  • ホームまたはアウェイ・チームの合計ポイント数が指定された数字を超えるか、越えないかを予想します。

マッチ・コレクト・スコアー

  • 特定の試合のフルタイムで正確な得点を予想します。

奇数 / 偶数

  • 特定期間でプレイした合計のゲーム数、セット数、ポイント数が偶数になるか、奇数になるかを予想します。
