ベットビルダー


一般規則
  • 特に明記されていない限り、すべてのフットボールのベットは、予定された「90分」のプレーに基づいて確定されます。
  • 「90分」プレーの期間は、どんな停止時間や怪我などの延長時間も含まれます。延長戦、ゴールデン・ゴール、ペナルティー・シュートアウトは、これに含まれず、また試合後に公式レフリーや関連した運営組織などによって試合結果が変更されたりした試合も含まれません。
  • ベットタイプの個々のルールに特に明記されてない限り、すべてのイン-プレイのベットは、予定された「90分」プレー終了時の結果に基づいて確定されます。
  • 「45分」プレーは、前半戦のプレーでのどんな停止時間や怪我などの延長時間も含まれます。延長戦、ゴールデン・ゴール、ペナルティー・シュートアウトは、これに含まれず、また試合後に公式レフリーや関連した運営組織などによって試合結果が変更されたりした試合も含まれません。
  • 特定された競技、または親善試合は80分（2 x 40分ハーフ）プレーが予定されており、ベットはその予定時間に基づいて確定されます。
  • 70分以下（2 x 35分ハーフ）で設定されている少年や親善試合の場合、当社は試合開始前に発表いたします。それ以外では、これらの試合にされた全てのベットは、無効になると考えられます。
  • 以下の個々のベットタイプ・ルールで特に明記がない限り、もし試合が公式「90分」プレー内で放棄された場合、すべてのベットは無効になると考えられます。
  • もし試合が前半戦の途中で放棄された場合、前半戦のベットはすべて無効になると考えられます。個々のベットタイプ・ルールに特に明記がない限り、もし試合が後半戦の途中で放棄された場合、すべての後半戦のベットは無効になります。すべての前半戦のベットは有効になります。
  • 個々のベットタイプ・ルールで特に明記されていない場合、オウンゴールはカウントされます。
  • もし予定された試合会場が変更された場合、もし本来アウェイのチームが現在ホームになっているような場合は、すべてのベットは無効になると考えられます。
  • 国際試合では、試合会場の変更が同じ国である限り、すべてのベットは有効になると考えられます。
  • 国際大会では、試合会場の変更が国内、または大会が本来予定された国で開催された場合、すべてのベットは有効になると考えられます。
  • もし他のシナリオでの試合会場の変更が試合結果に影響すると考えられた場合、当社はすべてのベットを無効にする権利を有しています。
  • 万が一、試合の正確な開始時間が分からない場合(例.TVスケジュールの影響など)、正式な開始時間から72時間以内になった時点で、我々は元の表示時間を調整する権利を有します。
  • 全てのベットが有効であると見なされるには、両チームが同じ数のプレイヤーで試合が開始される必要があります。

ハンディキャップ

一般規則

  • ハンディキャップが適応された試合/ハーフ/ピリオドで、誰が勝つかを予想します。
  • 「ハンディキャップ」の意味は、チームが架空的に有利なスタートを受け取り、試合が始まる前に得点差を付けることで試合を効果的にリードすることが出来ます。
  • ベットタイプ期間の終了時にハンディキャップが適応されることが要因になってベットが確定されます。
  • 好きなチームは、ハンディキャップ開始を与えて、マイナスのハンディキャップが割り当てられ、またハイライト表示されます。
  • ハンディキャップはフル・ゴール・スタート（例えば、-1, -2, -3,など）、ハーフ・ゴール・スタート（例えば、-0.5, -1.5, -2.5,など）またはスプリット・ゴール・スタート（例えば、-0/0.5, -0.5/1, -1/1.5,など）

ハンディキャップ

  • ハンディッキャップが適応された試合で誰が勝つかを予想します。
  • 「90分」プレーに基づいてベットは確定されます。
  • もし試合が「90分」を終了する前に放棄、停止、キャンセルまたは中止されると、すべてのベットは無効になると考えられます。

ハンディキャップ - 前半戦

  • すべての前半戦のベットは、前半戦のプレーにだけ適応されます。予定された「45分」終了時の得点でベットは確定されます。
  • もし試合でどんな理由があっても、前半戦の途中で放棄、停止、キャンセルまたは中止した場合、前半戦のすべてのベットは無効になると考えられます。
  • もし試合でどんな理由があっても、後半戦または延長戦の途中で放棄、停止、キャンセルまたは中止した場合、後半戦のすべてのベットは無効になると考えられます。

イン-プレイ・ハンディキャップ

  • ベットタイプ期間の終了時にハンディキャップが適応されることですべてベットが確定されます。
  • 結果の確定は、試合/ピリオドの終了時にベットされた時からのスコアラインに基づきます – すなわち、最終得点を現在の得点でマイナスします。前半戦ハンディキャップのベットは、前半戦の終了時の最終得点になります。

  • もし試合が延長戦の終了前に放棄、停止、キャンセルまたは中止した場合、すべてのベットは無効になると考えられます。

  • もしどんな理由があったとしても延長戦の前半戦の途中で試合が放棄、停止、キャンセルまたは中止した場合、すべての前半戦のベットは無効と考えられます。

ゴール – オーバー / アンダー

一般規則

  • 合計得点数が指定された数字を超えるか、超えないかを予想します。
  • もし合計得点数が指定された数字より多く得点されると、マーケットは「オーバー」として確定します。もし合計得点数が指定された数字より少なく得点されると、マーケットは「アンダー」として確定します。
  • ベットタイプ期間の終了時にハンディキャップが適応されることが要因となることで全てのオーバー/アンダーのベットは確定されます。
  • オーバー/アンダーのマーケットラインは、フル・ゴール（例えば、2, 3, 4,など）、ハーフゴール（例えば、1.5, 2.5, 3.5,など）またはスプリット・ゴール・ライン（例えば、1.5/2, 2.5/3, 3.5/4,など）になることが出来ます。
  • もし試合が放棄された場合、可能性のあるゴールがマーケット結果に影響しないと無条件に決められた時は、オーバー/アンダーのベットは確定されます。それ以外の場合、ベットは無効になると考えられます。以下の例をご覧下さい：
    • 例　1：メンバー・ベット・オーバー2.5ゴール
      • 得点が2-1の時に試合が放棄された場合
      • 試合が放棄されたにも関わらず、このマーケットは他のどの可能性のあるゴールもマーケット結果に影響しないと無条件に決められた場合、このメンバーは勝ちになります。
    • 例　2：メンバー・ベット・アンダー2.5ゴール
      • 得点が2 – 1の時に試合が放棄された場合
      • 試合が放棄されたにも関わらず、このマーケットがどんな可能性のあるゴールもマーケット結果に影響しないと決めた時は、メンバーのベットは失われます。
    • 例　3：メンバー・ベット・オーバー3.5ゴール：
      • 得点が2-1の時に試合が放棄された場合
      • マーケットの結果が無条件に決められる前に試合が放棄された場合、メンバーのベットは、無効になると考えられます。

  • もし試合が放棄された場合、指名した選手がまだ得点していなければ無効になると考えられます。しかし、もし指名した選手が試合放棄前にレッドカードを受けていたら、その選手に関連したどんなベットも負けベットになると考慮されます。
  • 延長戦またはペナルティー・シュートアウトの途中で得点されたオウンゴールやゴールはカウントされません。

ゴール・スコア・ヘッド・トゥ・ヘッド

ヘッド・トゥ・ヘッド(一般規則 )

  • ファースト・ハーフ（前半戦）のベットは「45分」にだけ適応できます。
  • フルタイムのベットは「90分」を通して確定されます。
  • ベットが確定するためには、両参加者が試合の先発の11人に入っていなければなりません。
  • マーケットは、指定された日の選手が参加した試合にだけ適応します。
  • オウンゴールは「ゴール・スコアラー・ヘッド・トゥ・ヘッド」としてカウントされません。

ヘッド・トゥ・ヘッド (ハンディキャップ)

  • ハンディキャップが適応された後に、もっとも多くゴールを得点する選手を予想します。
  • もし試合が放棄または延期された場合、可能性のあるゴールがマーケット結果に影響しないと無条件に決められなければ、すべてのベットは無効になると考えられます。

ヘッド・トゥ・ヘッド (オーバー / アンダー)

  • 両選手による合計得点数が適応されたラインをオーバー（超える）するか、アンダー（超えない）するかを予想します。
  • もし試合が放棄、または延期された場合、可能性のあるゴールがマーケット結果に影響しないと無条件に決められなければ、すべてのベットは無効になると考えられます。

  • どの選手がもっとも多く得点するかを予想します。ドロー/タイもベッティング・オプションにはいります。
  • もし試合が放棄または延期された場合、どんなゴールもマーケット結果に影響しないと決められなければ、すべてのベットは無効になると考えられます。

ヘッド・トゥ・ヘッド (奇数 / 偶数)

インジャリー・タイム・アワード・アット・ザ・エンド・オブ・ザ・2nd・ハーフ – オーバー / アンダー（後半戦の終了時に追加される時間）

ペナルティー・マーケット

一般ルール

  • パーレイ・シュートアウト・マーケットの精算は、勝者が決まった際のラウンド(およびペナルティー)に基づくものとします。
  • 試合ルールは、勝者が決定次第、精算のため、全てのペナルティーは無視されます。

ペナルティー・アワード

  • 結果に関係なく、フル「90分」プレー内で、ペナルティーが与えられるかを予想します。

PK戦 - ハンディキャップ

  • 指定のハンディキャップが適応されたペナルティー・シュートアウトで誰が勝利するかを予想します。
  • サドンデス（延長戦）は、ペナルティー・シュートアウトのハンディキャップ・ベッティングに含まれます。
  • もし試合がシュートアウトにならなかった場合、すべてのベットは無効になります。
  • 確定を目的とした場合、ペナルティー・スコアは「90分」プレーを通したものと、また延長戦の期間は含まれません。

PK戦 -ゴール: オーバー / アンダー

  • シュートアウトのペナルティー・スコアの合計数が指定された数字を超えるか（オーバー）、超えないか（アンダー）を予想します。
  • ペナルティー・シュートアウトのオーバー/アンダー・ベッティングは、シュートアウトの基本のペナルティーキック10回（各チーム5回づつ）だけが含まれます。サドンデス・ペナルティーは含まれません。
  • 例：
    • リバプール4-1トッテンハム – オーバー/アンダーのラインは5で確定します。
    • リバプール6-5トッテンハム（各5ペナルティーキック後の結果：リバプール4-4トッテンハム）- 各チームが5回づつのペナルティーキックを蹴った後、ペナルティースコアの合計数として、オーバー/アンダーのラインが8で確定します。
  • もし試合がシュートアウトにならなかった場合、すべてのベットは無効になります。
  • 確定を目的とした場合、ペナルティー・スコアは「90分」プレーを通したものと、延長戦の期間は含まれません。
  • もしペナルティー・シュートアウトの間に試合が放棄された場合、どんなペナルティー・スコアもマーケット結果に影響しないと無条件に決められた時だけ、オーバー/アンダーのベットは確定されます。それ以外の場合は、無効になると考えられます。

延長時間 – ウィニングマージン

  • 延長時間の終了時に、勝敗チームと負けたチームを分けるゴールの数を予測する。
  • ベットは、「30分」延長時間プレイ（怪我による中断時間を含む）に基づいて決済され、その後のPK戦は含まれない。
  • ドロー/タイとして終了するマッチは、「スコアドロー」または「ゴールなし」にとして決済されます

ペナルティー（シュートアウト） - ゴール / ノーゴール

  • チームが特定のペナルティーシュートアウト（PK）で得点するかを予想します。
  • ゴールまたはノーゴールはペナルティーの結果発表に基づいて確定されます。

PK戦

  • 特定の試合がPK戦に進むかどうかを、「はい」または「いいえ」と予測します。
  • 決済は、余分な時間をプレイすることができるかどうかにかかわらず、PK戦に進むゲームに基づいて行われます。

PK戦 – ラウンドX

  • ラウンドXに勝つかどうか、または結果が引き分けになるかどうかを予測する。
  • ラウンドXは、ラウンドの両チームのペナルティの合計結果として定義される。

    • 例） （PK戦の開始） - チームAの第1ペナルティキックとチームBの第1ペナルティキック=ラウンド1。
    例） （サドンデスの開始） - チームAの6回目のペナルティキックとチームBの6回目のペナルティキック=ラウンド6。

  • 決済は、各ラウンドごとに各チームが記録したゴール数に基づいて行われます。
  • PK戦に進まない場合、すべてのベットは無効になります。

PK戦 – 最終ラウンド

  • PK戦が終了するラウンドを予測する。
  • 決済は、第3、第4、第5、または6（またはそれ以降）のPK戦の試合終了に基づいて行われます。
  • 結果が発表される前に試合が放棄された場合、このマーケットのベットは無効とみなされます。

    • -もし延期または放棄された時の合計得点が指定された「ライン」を超えた場合、ベットは有効と考慮されます。

    -もし延期または放棄された時の合計得点が指定された「ライン」を超えなかった場合、ベットは無効になると考慮されます。

ダブルチャンス & ゴール・オーバー / アンダー

  • 正確な結果の選択（以下で明記された）、それと同時に試合内で両チームがゴールを記録するかに基づいて確定されます。

トータル・ホーム＆アウェイ・チーム・イン・ア・パーティキュラー・リーグ（特定のリーグでホームとアウェイチームの合計）

特定のリーグで、組み合わされたイベントの結果を決めるために、イベントを通してメンバーにベットをする機会を提供しております。このマーケットは、すべてのアウェイチームの結果に対してすべてのホームチームの結果を組み合わせたものです。中立の試合では、このマーケットのチーム1が「ホームチーム」と考えられます。以下は、当社が提供する個々のベットタイプのルールの内訳となります。


特定リーグでのホーム戦とアウェイ戦の合計：1 x 2、ダブルチャンス

    • ”アウェイチーム” (1 x 2)
    • ”アウェイチーム / ドロー”と”ホームチーム / アウェイチーム” (ダブルチャンス)

ファンタジー・マッチ

2つの異なる試合から1チームずつ選び、その2チームが対戦した場合の架空の結果を予想します。

  • ファンタジー・マーケットの結果は、それぞれのチームがそれぞれの試合で得点したゴール数により決定されます。
  • どちらかの試合(または両方の試合)が棄権、中止、延期となり、予定されていた開始時間から36時間以内に再開できない場合、ファンタジー・マッチへの全ベットは無効となります。
  • どちらかの試合(または両方の試合)が延長戦やPK戦に突入した場合、90分間での試合結果（ロスタイムを含む）のみが対象となります。
  • ファンタジー・マッチにおけるホーム又はアウェイは、試合結果やベットの確定に影響はありません。
  • ファンタジー・マーケットで提供されているベットの確定は、上記に記載されている特定のマーケットルールに従います。

ベットビルダー

  • ベットビルダーは、選択されたイベントとマーケットタイプでのみご利用いただけます。いつでもイベントから除外もしくは一時的に無効になる可能性があります。
