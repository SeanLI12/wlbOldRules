Regras Gerais
  • Se o local previsto for alterado, todas as apostasy serão consideradas nulas.
  • As regras de empate se aplicam para fins de liquidação.
  • Se um torneio termina oficialmente antes do prazo estipulado e o troféu é concedido, todas as apostas são consideradas válidas. As apostas aceitas após o fim do jogo, são consideradas nulas, a menos que tenha havido uma jogada que contribuiu para o resultado do torneio (playoffs não incluida).
  • Se o torneio é oficialmente abandonado, todas as aposta são canceladas, exceto mercados que ainda não foram decididos.
TIPOS DE APOSTAS

Mercado deCampeão (Vencedor)

  • Prevê quem vai ganhar o torneio. Apostas são resolvidos com base em que jogador ganha o troféu.
  • Sem competição, sem aposta. Caso um jogador se retire antes de bater no primeiro round, então todas as apostas neste jogador serão anuladas e a aposta devolvida.
  • Considera-se que um jogador começou se tiver dado uma tacada. Se um jogador posteriormente se retira ou é desqualificado, então todas as apostas serão válidas.
  • "Qualquer jogador não listado" refere-se à todos os jogadores não mencionados no Mercado de Campeão.
  • Mercados de Campeão será determinado com base no vencedor do torneio e caso de desempate (play-off) são levados em consideração o resultado final.

Head to Head / 2 Bolas / 3 Bolas (18 Buracos)

  • Prevê quem vai completar a rodada na menor pontuação.
  • Jogadores estão emparelhados para propósitos de apostas. Eles podem ou não podem jogar juntos na competição real.
  • O vencedor sera o jogador com a menor pontuação em 18 buracoes..
  • Se um jogador falhar no começo da rodada, todas as apostas serão consideradas nulas.
  • Ambos os jogadores devem completar 18 buracos por rodada de aposta. Se um dos jogadores terminar antes por acordo dos jogares ou por lesão, todas as apostas serão consideradas nulas.
  • Para 2 bolas, o empate sera uma selação de apostas.
  • Para 3 bolas, o empate não sera uma seleção de apostas. Em vez disso, as regras de eliminação se aplicam para o caso de empate.

Head to Head / 2 Bolas / 3 Bolas (72 Holes / Tournament Match Bet)

  • Prever quem vai completar o torneio com a pontuação mais baixa.
  • Os jogadores estão emparelhados com a finalidade de apostar. Eles podem ou não podem jogar juntos na competição real.
  • O vencedor srá o jogador com a menor pontuação em 72 buracos, incluindo quaisquer buracos de play-off.
  • Se um jogador falhar com começo da rodada, todas as apostas serão consideradas nulas.
  • Ambos os jogadores devem completar todos os 72 buracos para combinação de apostas do torneio.
  • Para 2 bolas, o empate sera uma seleção de apostas.
  • Para 3 bolas, o empate não será uma seleção de apostas. Em vez disso, as regras de empate se aplicam no caso de empate.

Próxima pontuação de buraco – Mercado: Único jogador

  • Prever o próximo acerto de buraco feito por um jogador ou jogador mencionado. Prever (Birdie or melhor / Par / Bogey or pior)
  • Se o jogador falhar no inicio para acertar o respectivo buraco, então as apostas serão consideradas nulas.
  • Uma vez que o jogador tenha dado uma tacada para acertar o buraco indicado, as apostas serão válidas. Se um jogador falhar ao dar uma tacada no buraco (lesão ou retirada etc), então o “over-par” será premiado.
  • O pagamento da aposta sera baseado na pontuação oficial após o término do jogo, como registrado pelo órgão competente (European Tour / PGA)

Handicap

  • Prevê quem vai completar a rodada / torneio na menor pontuação. Um começo virtual é dado para um jogador. Se ambos os jogadores têm a mesma pontuação de tomar a desvantagem em consideração, será considerado um empate e todas as apostas serão devolvidas.

Acima / Abaixo (Placar)

  • Prevê se o resultado final de ambos os jogadores se estão acima ou abaixo da linha total indicada.
  • No caso de abandono, apostas neste mercado serão resolvidas apenas quando o Mercado tiver sido determinado incondicionalmente. Em todos os outros cenários, as apostas serão consideradas nulas.

Top 5 / 10 Finish

  • Prevê se o jogador vai terminar entre os 5 / 10 primeiros no torneio específico.
  • Aplicar as regras do Dead-Heat.
  • Para fins de apostas, as mesmas serão pagas no final do torneio programado.

Margem de Vitória

  • Prever a diferença do número do grau de dificuldade de cada buraco (Strokes) entre o vencedor e o vice-campeão (runner-up) no final do torneio.
  • Para fins de apostas, as apostas serão pagas no final do torneio programado.
  • No caso de jogadas extras para desempate play-off, os resultados dos mesmos não serão contabilizados para o resultado final.
